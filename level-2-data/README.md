# level-2 — excel + xml

Скрипт для оновлення цін і вигрузки в XML.

## Файли

В `data/`:
- `Import.xlsx` — вигрузка каталогу
- `Прайс.xlsx` — прайс постачальника (багато листів, різні формати)

Після запуску в `output/`:
- `Import_оновлений.xlsx`
- `catalog.xml`

## Як запустити

```bash
pip install -r requirements.txt
python3 process.py
```

Якщо файли в іншому місці:
```bash
python3 process.py --import шлях/до/Import.xlsx --price шлях/до/Прайс.xlsx
```

## Що робить

1. Парсить прайс — там не один формат, тому `price_loader.py` обходить всі листи (Woks, ExTherm, Devi, Ryxon і т.д.)
2. Зіставляє по артикулу і підтягує нову ціну в імпорт
3. `Старая цена` = ціна * 1.10
4. Жовтий рядок — ціна змінилась, зелений — лишилась як була
5. XML пишеться порціями по 500 офферів, щоб нормально тягнуло 10к+ товарів

## Структура

- `process.py` — основна логіка
- `price_loader.py` — витягування цін з прайсу
- `generate_samples.py` — тестові файли, якщо треба поганяти без реальних даних

На реальних файлах вийшло ~2059 товарів, оновилось 68 позицій.
